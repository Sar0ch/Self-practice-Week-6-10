Week6
- เข้าใจแนวคิดของ First-Class 
- เข้าใจ Higher-Order Function
- Pure Function = ฟังก์ชันที่ input เดิม และ output ก็ต้องเหมือนเดิมเสมอ
- Immutability = ไม่แก้ไขข้อมูลต้นฉบับ แต่สร้างของใหม่แทน
- Function Composition คือการเอาฟังก์ชันเล็ก ๆ มาต่อกัน
ทำให้เข้าแนวคิดการเขียนฟังก์ชั่นมากขึ้น แต่แบบฝึกหัดที่ทำยังรู้ว่าไม่เข้าใจมากเท่าไหร่ ยังงงๆอยู่บ้างต้องให้เพื่อนช่วย guide และ บางส่วนต้องถาม chatgpt เพื่อให้มันช่วยแก้

Week7
- เข้าใจหลักการของ DOM
- เข้าใจการเข้าถึงและตรวจสอบ Node
- ฝึกเข้าถึงลูกและ attribute ของ element
- เข้าใจการสร้างและเพิ่ม node ใหม่ลงใน DOM
- ได้ลองเขียนโค้ดหลายแบบทำให้เข้าใจมุมกว้างมากขึ้น 

Week8

- เข้าใจการ CRUD nodes มากขึ้น สามารถทำงานกับมันได้ดีขึ้น
- ยังงงๆเรื่อง dataset กับ classList บ้างแต่พยายามใช้ chatgpt อธิบายเพิ่มขึ้นแล้ว
